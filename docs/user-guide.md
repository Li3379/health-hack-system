## 1. 快速开始

### 1.1 系统要求

**浏览器要求：**
- Chrome 80+（推荐）
- Firefox 75+
- Safari 13+
- Edge 80+

**网络要求：**
- 稳定的网络连接
- 支持 WebSocket 连接（用于实时监控功能）

**推荐设备：**
- 桌面端：屏幕分辨率 1280x720 或更高
- 移动端：支持响应式布局，建议使用最新版浏览器

### 1.2 注册账户

1. 访问系统首页，点击「立即注册」链接
2. 填写注册信息：
   - **用户名**：4-32个字符，用于登录
   - **密码**：6-64个字符，建议包含字母、数字和特殊字符
   - **确认密码**：再次输入密码确认
   - **邮箱**：可选，用于找回密码和接收通知
3. 点击「立即注册」按钮完成注册

**注意事项：**
- 用户名一旦注册不可修改
- 密码长度至少6位，建议使用强密码
- 邮箱格式需正确，如：user@example.com

### 1.3 登录系统

1. 访问系统首页
2. 输入用户名和密码
3. 点击「登录」按钮

**登录提示：**
- 登录凭证有效期为7天，期间无需重复登录
- 若忘记密码，请联系管理员重置

### 1.4 仪表板概览

登录后进入系统仪表板，主要包含以下区域：

**顶部区域：**
- 欢迎信息：显示当前用户昵称
- 快捷操作：添加指标、AI咨询按钮

**统计卡片：**
- 健康评分：综合健康评估分数及趋势
- 未读预警：待处理的健康预警数量
- 健康指标：已记录的健康数据总数
- 体检报告：上传的体检报告数量

**快速入口：**
- 添加指标：快速录入健康数据
- AI 顾问：进入AI健康咨询
- 上传报告：上传体检报告
- 实时监控：查看实时健康数据
- 保健中心：保健指标管理

## 2. 快捷录入

快捷录入功能支持手动输入各类健康和保健指标数据。

### 2.1 进入快捷录入

1. 点击左侧导航栏「数据录入」
2. 选择「快捷录入」卡片
3. 点击需要录入的指标按钮

### 2.2 支持的指标类型

**健康指标：**

| 指标名称 | 单位 | 有效范围 | 说明 |
|---------|------|---------|------|
| 血糖 | mmol/L | 0-50 | 空腹或餐后血糖值 |
| 血压 | mmHg | 收缩压 60-250，舒张压 40-200 | 需同时输入收缩压和舒张压 |
| 心率 | 次/分 | 20-300 | 静息心率 |
| 体温 | C | 30-45 | 体温测量值 |
| 体重 | kg | 20-300 | 体重测量值 |
| BMI | - | 0-100 | 体质指数 |

**保健指标：**

| 指标名称 | 单位 | 有效范围 | 说明 |
|---------|------|---------|------|
| 睡眠 | 小时 | 0-24 | 睡眠时长 |
| 步数 | 步 | 0-100000 | 每日步数 |
| 饮水 | ml | 0-10000 | 每日饮水量 |
| 心情 | - | 1-10 | 心情评分 |
| 精力 | - | 1-10 | 精力状态评分 |
| 运动 | 分钟 | 0-1440 | 运动时长 |

### 2.3 录入步骤

1. 点击对应指标按钮，弹出录入对话框
2. 输入数值（系统会自动验证范围）
3. 选择记录日期（默认为当天）
4. 点击「保存」完成录入

**血压录入特殊说明：**
- 血压需同时输入收缩压（高压）和舒张压（低压）
- 系统会自动保存两个指标

### 2.4 数值验证规则

系统会对输入的数值进行实时验证：

- 数值必须在有效范围内
- 小数精度：血糖、体温、体重、BMI 支持一位小数
- 整数精度：心率、血压、步数等必须为整数

若输入超出范围，系统会提示正确的数值范围。

## 3. AI 智能录入

AI 智能录入支持使用自然语言描述健康数据，系统自动解析并提取指标。

### 3.1 使用方法

1. 进入「数据录入」页面
2. 在「AI 智能录入」卡片中输入描述文字
3. 点击「智能解析」按钮
4. 查看解析结果，选择需要保存的指标
5. 点击「确认录入」完成保存

### 3.2 支持的输入格式

**单指标输入：**
```
今天血糖5.6
心率72
体重65公斤
```

**多指标输入：**
```
今天血糖5.6，心率72，走了8000步
早上血压120/80，血糖空腹6.2
```

**带时间描述：**
```
昨天睡眠7小时
前天步数10000步
```

### 3.3 解析结果说明

解析结果包含以下信息：

- **指标名称**：识别出的健康指标
- **数值**：提取的数值及单位
- **置信度**：解析可信程度
  - 绿色（90%+）：高置信度，建议直接保存
  - 黄色（70-90%）：中等置信度，建议核对后保存
  - 红色（<70%）：低置信度，建议手动修改

### 3.4 每日使用限制

- AI 智能录入每日有使用次数限制
- 界面会显示「今日剩余」次数
- 次数用完后需次日重置

**限制原因：**
- AI 解析需要调用外部服务
- 为保证服务稳定性，设置每日限额

## 4. OCR 图片识别

OCR 图片识别支持上传健康相关图片，自动识别并提取健康数据。

### 4.1 支持的图片类型

**图片格式：**
- JPG / JPEG
- PNG
- WebP

**文件大小限制：**
- 最大 10MB

### 4.2 支持的识别类型

| 类型 | 说明 | 示例 |
|-----|------|------|
| 体检报告 | 医院体检报告单 | 血常规、生化检查等 |
| 药品标签 | 药品包装或说明书 | 药品名称、用法用量 |
| 营养标签 | 食品营养成分表 | 热量、蛋白质、脂肪等 |

### 4.3 使用步骤

1. 进入「数据录入」页面
2. 在「OCR 图片识别」卡片中选择识别类型
   - 体检报告
   - 药品标签
   - 营养标签
3. 拖拽或点击上传图片
4. 点击「开始识别」按钮
5. 查看识别结果：
   - 检查识别的指标是否正确
   - 可取消勾选不需要的指标
   - 点击「原始识别文本」查看完整识别内容
6. 点击「确认录入」保存数据

### 4.4 识别结果解读

**识别状态：**
- 识别成功：绿色标签
- 识别失败：红色标签（可能原因：图片模糊、文字不清晰等）

**置信度说明：**
- 90%+：识别准确，可直接保存
- 70-90%：建议核对数值
- <70%：建议手动修正

**注意事项：**
- 上传清晰的图片可获得更好的识别效果
- 图片文字应水平放置，避免倾斜
- 确保光线充足，避免阴影遮挡

## 5. 设备同步

设备同步功能支持连接智能穿戴设备，自动获取健康数据。

### 5.1 支持的平台

| 平台 | 设备类型 | 同步数据 |
|-----|---------|---------|
| 华为健康 | 华为手环、手表 | 步数、心率、睡眠、血压等 |
| 小米运动 | 小米手环、手表 | 步数、心率、睡眠等 |
| 微信运动 | 手机计步 | 步数 |
| Apple Health | iPhone、Apple Watch | 步数、心率、睡眠等 |

*注：实际支持平台以系统显示为准*

### 5.2 连接设备

1. 进入「数据录入」页面
2. 在「穿戴设备同步」卡片中找到目标平台
3. 点击「连接」按钮
4. 在弹出的授权窗口中完成 OAuth 授权
5. 授权成功后，设备状态将变为「已连接」

**授权说明：**
- 首次连接需要跳转到对应平台进行授权
- 授权后系统可读取设备数据
- 可随时断开连接撤销授权

### 5.3 同步数据

**单设备同步：**
1. 找到已连接的设备
2. 点击「同步」按钮
3. 等待同步完成
4. 查看同步结果（同步的数据条数）

**全部同步：**
- 点击「同步全部」按钮
- 系统将依次同步所有已连接设备
- 显示各设备同步状态

### 5.4 同步状态说明

| 状态 | 说明 |
|-----|------|
| 已连接 | 设备已授权，可正常同步 |
| 未连接 | 设备未授权，需先连接 |
| 同步中 | 正在获取数据 |
| 同步成功 | 数据已成功导入 |
| 同步失败 | 请检查网络或重新授权 |

### 5.5 查看同步历史

1. 点击「查看同步历史」按钮
2. 查看历史同步记录：
   - 同步平台
   - 数据条数
   - 同步状态
   - 同步时间
   - 耗时

## 6. 常见问题

### 6.1 登录问题

**Q: 忘记密码怎么办？**
A: 请联系系统管理员重置密码。后续版本将支持邮箱找回功能。

**Q: 登录后提示会话过期？**
A: 登录凭证有效期为7天，超时后需重新登录。

**Q: 无法登录，提示密码错误？**
A: 请检查：
- 用户名是否正确
- 密码大小写是否正确
- 输入法是否为英文模式

### 6.2 数据录入问题

**Q: 数值输入提示超出范围？**
A: 请检查输入值是否在有效范围内。各指标范围请参考[第2.2节](#22-支持的指标类型)。

**Q: 血压录入保存失败？**
A: 血压需同时输入收缩压和舒张压，请确保两项都已填写。

**Q: 录入的日期可以修改吗？**
A: 可以。在录入对话框中选择正确的日期即可。支持录入过去的健康数据。

### 6.3 AI 智能录入问题

**Q: AI 解析失败怎么办？**
A: 可能原因及解决方案：
- 输入描述不够清晰：尝试更明确地描述，如「血糖5.6」而非「血糖正常」
- 服务暂时不可用：稍后重试
- 每日次数用完：次日再试

**Q: 解析结果不准确？**
A: 可取消勾选不准确的指标，或使用快捷录入手动输入。

**Q: 为什么有每日次数限制？**
A: AI 解析依赖外部服务，为保证服务稳定性和公平使用，设置每日限额。

### 6.4 OCR 识别问题

**Q: 图片上传失败？**
A: 请检查：
- 图片格式是否为 JPG、PNG 或 WebP
- 图片大小是否超过 10MB
- 网络连接是否正常

**Q: 识别结果不准确？**
A: 建议：
- 上传清晰的图片
- 确保文字水平放置
- 避免光线不足或有阴影
- 手动修正后保存

**Q: 支持手写内容识别吗？**
A: 当前主要支持印刷体识别，手写内容识别效果可能不佳。

### 6.5 设备同步问题

**Q: 连接设备时授权页面打不开？**
A: 请检查：
- 浏览器是否阻止了弹出窗口
- 网络连接是否正常
- 尝试使用其他浏览器

**Q: 同步失败怎么办？**
A: 可能原因：
- 设备授权已过期：断开连接后重新授权
- 网络问题：检查网络后重试
- 设备平台服务异常：稍后重试

**Q: 同步的数据在哪里查看？**
A: 同步成功后，数据会自动保存到对应的健康指标中，可在「健康指标」页面查看。

**Q: 可以同步历史数据吗？**
A: 设备同步会获取最近的健康数据，历史数据同步范围取决于设备平台的支持情况。

### 6.6 其他问题

**Q: 如何修改个人信息？**
A: 点击右上角用户头像，进入「个人设置」页面进行修改。

**Q: 数据安全吗？**
A: 系统采用以下安全措施：
- 数据传输使用 HTTPS 加密
- 密码使用 BCrypt 加密存储
- JWT 令牌认证机制
- 敏感数据不对外暴露

**Q: 遇到其他问题怎么办？**
A: 请联系系统管理员或技术支持团队。
