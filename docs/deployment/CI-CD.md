# CI/CD 流程

> 本文档描述 HHS 项目的持续集成和持续部署流程，基于 GitHub Actions 实现自动化构建、测试与部署。

## 流程概览

```
┌────────────────────────────────────────────────────────────────────────────┐
│                         CI/CD 完整流程                                    │
├────────────────────────────────────────────────────────────────────────────┤
│                                                                          │
│  代码推送                                                                 │
│  ────────                                                                │
│     │                                                                    │
│     ├─ hhs-backend/** ──► Backend CI ──► 构建验证 + 集成测试              │
│     │                                              │                     │
│     ├─ hhs-frontend-v2/** ──► Frontend CI ──► 构建 + Lint + 测试        │
│     │                                              │                     │
│     └─ push to master ──► Build & Deploy ──► Docker 镜像构建 ──► 自动部署│
│                                  │                  │                     │
│                                  │           GHCR (ghcr.io)              │
│                                  │                  │                     │
│                                  └──────► Deploy (手动) ──► 生产环境部署  │
│                                           需审批通过                      │
└────────────────────────────────────────────────────────────────────────────┘
```
